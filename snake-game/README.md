# Classic Snake Game

A simple classic Snake implementation in plain HTML/CSS/JavaScript.

## How to Run

1. Open `snake-game/index.html` in a browser.
2. Use arrow keys or **WASD** to move.
3. Eat the food (red/orange dot) to grow. Avoid hitting the walls or your tail.

## Files

- `index.html` — markup and layout
- `styles.css` — styling for board, snake, and UI
- `script.js` — game logic and input handling
